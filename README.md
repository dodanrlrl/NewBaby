## 게임 소개

- 게임 이름 : Babyjack - Isaac 모방

- 구현 사항
  - 캐릭터
   - 조작 (이동, 공격)
   - 스텟
  - 아이템
   - 인벤토리
   - 아이템 드롭
   - 아이템 효과
  - 맵
   - 랜덤 맵 - 난이도 상 여러 맵 중 순서를 랜덤으로 설정하도록 변경
   - 퍼즐 요소
  - 몬스터
   - 피해와 체력 관리 - 생성과 파괴
   - 전투
   - 퍼마데스(캐릭터 죽음 == 게임종료)
   - 드랍 아이템
   - 보스 몬스터
